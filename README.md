# Requirements
- Maven 3.6.1 and Java 8 must be installed in the computer.

# To run the DiscountCalculationDemoTest
To run it, just do left click in the green arrow present at the same line of the main method, and select
the first option.

# To run the test classes under test directory
- If you want to run all the test at once, just right click on the test class and select "Run '<class_name>"
- Otherwise, to run the tests separately, open the class and click on the green arrow at the left of
  the test name and select the first option.
